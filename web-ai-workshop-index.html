Realita

      
Čomu veriť a čo si overiť

      

        

          
Limity AI

          
!Mýli sa sebavedomo. Halucinácie - overte fakty, čísla aj zdroje.

          
!Nepozná úplne najnovšie veci. Má „knowledge cutoff".

          
!Vymyslí si aj kód. Funkciu či knižnicu, ktorá neexistuje - vždy odskúšajte.

        

        

          
Vaša zodpovednosť

          
✓Výstup je váš. Ručíte zaň vy, nie AI.

          
✓Buďte transparentní, kde to dáva zmysel.

          
✓Myslite na súkromie. Osobné údaje a GDPR pri formulároch.

        

      

      
Pravidlo: AI je skvelý prvý draft, vy ste posledné slovo.

Remeslo

      
Ako vyzerá dobrý prompt

      

        

          
Slabý prompt

          
Sprav mi web pre moju kaviareň.

          
Žiadny cieľ, žiadna cieľová skupina, žiadny tón. AI háda - a výsledok je priemerný.

        

        

          
Silný prompt

          
# Cieľ
One-pager pre malú kaviareň v Bratislave.
Cieľ: získať rezervácie.
# Cieľová skupina
Ľudia 25-40, hľadajú kvalitnú kávu.
# Sekcie
Hero, o nás, ponuka, hodiny, kontakt.
# Dizajn
Teplé farby, moderný font, čisté HTML.

          
          

            
✓ Hotový prompt - rovno použiteľný

            
Vytvor jednostránkový web (one-pager) pre novú kávu „Ranný Štart".
Cieľ: získať registrácie do predpredaja cez jedno tlačidlo (CTA).
Skupina: ľudia 25-40 v Bratislave, čo chcú kvalitnú kávu domov.
Sekcie: hero + CTA, o produkte, ponuka a cena, recenzie, FAQ, kontakt.
Dizajn: teplá hnedá a krémová, moderný font, priateľský tón.
Technika: čisté HTML + CSS, responzívne, jeden súbor, bez backendu.

Pravidlá hry

      
Čo nerobiť a čo áno

      

        

          
Vyhnite sa (zatiaľ)

          
✕Backend - serverová logika sa ťažko stráži a zabezpečuje.

          
✕Admin - prihlasovanie a správa používateľov = riziko.

          
✕Platobné brány - peniaze patria do overených riešení.

          
✕Vlastné databázy - nesprávne uložené dáta sú bezpečnostné riziko.

        

        

          
Pokojne robte

          
✓Statické stránky - čisté HTML & CSS. Rýchle a bezpečné.

          
✓Kontaktný formulár cez overenú službu (napr. Tally).

          
✓Responzívny dizajn - funguje na mobile aj počítači.

          
✓Nasadenie cez Vercel - HTTPS a hosting zadarmo.

        

      

      
Pravidlo: čím menej pohyblivých častí, tým menej toho, čo sa môže pokaziť alebo zneužiť.

    

  


  
  

    

      
Dva svety

      
Lokálne vs. nasadené

      

        

          
Lokálne (váš počítač)

          
  • Stránku vidíte len vy
  • Ideálne na tvorbu a testovanie
  • Nič nie je verejné

        

        

          
Nasadené (internet)

          
  • Stránka má vlastnú adresu (URL)
  • Dostupná komukoľvek na svete
  • Automatické HTTPS a rýchle načítanie

        

      

      
Postup: vytvorím lokálne → uložím → nasadím. Nástroje na to (Cursor · GitHub · Vercel) sme už spomenuli.

Prompt 3 zo 4

      
Štruktúra landing page

      

        
## Sekcie zhora nadol
1. Hero - názov eventu, dátum, miesto, CTA „Kúpiť lístky"
2. Odpočet (countdown) do začiatku
3. O evente - čo ťa čaká
4. Program / line-up
5. Miesto a ako sa tam dostať
6. Lístky a cena + CTA
7. FAQ
8. Pätička - kontakt a sociálne siete

## Technika
Čisté HTML + CSS + trochu JS (odpočet),
responzívne, jeden súbor.

        

          
    
            
  • Jedna stránka = jeden cieľ. Všetko vedie k „Kúpiť lístky".
    • 
            
  • CTA hore aj dole. Návštevník nemusí hľadať.
    • 
            
  • Menej je viac. Každá sekcia má jeden účel.
    • 
            
  • Štruktúru pokojne upravte podľa produktu - toto je dobrý štart.
